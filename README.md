一个基于 Lazyvim 的 Neovim 配置
使用方法：
    1. 安装 Neovim
    2. 执行：
        Windows: `git clone --depth=1 https://gitee.com/Sigechaishijie/VimX $env:LOCALAPPDATA\nvim && Remove-Item $env:LOCALAPPDATA\nvim\.git -Recurse -Force`
        Unix-Like: `git clone --depth=1 https://gitee.com/Sigechaishijie/VimX ~/.config/nvim && rm -rf ~/.config/nvim/.git`
